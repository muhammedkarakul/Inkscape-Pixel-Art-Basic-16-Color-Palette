# Inkscape Pixel Art 16 Basic Color Palette

## How To Use?

Steps
 - Clone the project.
 - Copy and Paste pixelArt.gpl files to Inkscape plattes folder.(path for mac os x: I/Applications/Inkscape.app/Contents/Resources/share/inkscape/palettes )
 - Make some pixel art!
